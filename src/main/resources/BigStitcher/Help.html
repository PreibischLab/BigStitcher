Key Bindings for BigStitcher

When the BigDataViewer window is active (right click), it will display those image stacks that are currently selected in the Stitching Explorer. While the Stitching Explorer is in focus, you can use special keys to change the way the images are displayed.

When the BigDataViewer window is active, it has special key-bindings that allow you to browse and display the data. Please press F1 then to display another help window with more details.


C Switch between fused mode and random coloring of image stacks.
L Display links between images used for registation (gray=weak links, orange=filtered link, red=inconsistent link, green=strong link).
R Re-orient BigDataViewer onto the currently selected tiles.
